Readme.txt for FIB.ASM (Fibonacci sequence)

Status: Functional

To run: 
		Download Misasim
		Load the FIB.ASM file
		Run

Objective:
		Learn assembly (MIPS) by writing Fibonaccis sequence.
		Finds the n th fibonacci number
